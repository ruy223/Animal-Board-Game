# Animal-Board-Game
AnimalGame is a Python-based abstract strategy board game engine built around object-oriented design and rule-driven gameplay. The game takes place on a 7x7 board where two players, Tangerine and Amethyst, control animal-themed pieces with unique movement patterns, including both sliding and jumping mechanics, directional restrictions, capture rules, and special movement exceptions. The project uses a base Piece class with specialized subclasses for each piece type (Pika, Trilobite, Wombat, and Beluga), allowing movement logic to be encapsulated through inheritance and polymorphism. The main AnimalGame class manages board state, validates legal moves, enforces turn order, handles captures, and detects victory conditions when a player’s Beluga is captured.

This project was designed to demonstrate core software engineering concepts through a non-trivial rules engine, including object-oriented programming, grid-based state management, algorithmic move validation, and game-state control. Players interact through algebraic board coordinates (such as b2 to b4), while the engine determines whether a move is legal based on piece behavior, obstructions, turn ownership, and capture logic. Beyond implementing the game itself, the project emphasizes clean class design, modular logic, and solving complex movement constraints in code, making it both a functional game engine and a portfolio piece showcasing Python development and problem-solving skills.
